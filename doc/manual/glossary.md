# Glossary

| Term | Meaning |
|---|---|
| OCN | MAPIR filter set with Cyan / Orange / NIR1 bands. |
| RGN | MAPIR filter set with Red / Green / NIR2 bands. |
| NGB | MAPIR filter set with NIR2 / Green / Blue bands (camera-dependent). |
| NIR1 / NIR2 | Two near-infrared filter options used by Survey3 variants. |
| FOURCC | 4-character code identifying pixel format (e.g., `MJPG`, `H264`). |
